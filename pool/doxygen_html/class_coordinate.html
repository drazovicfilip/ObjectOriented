Detailed Description

The Coordinate class is used for any balls in the program.

Author
Filip Drazovic
Date
April 2018

Balls are given a coordinate when they are initialised, and this coordinate is updated as the program runs. Every ball also stores the table dimensions, which can be useful for calculating collisions.
