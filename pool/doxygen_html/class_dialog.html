nextFrame will render the next frame using each object's render method

This will call the pool class' render method, which will use the render method of each ball and table. The rendering rate (framerate) can be changed by altering the frametime variable

void Dialog::simulate ( )

simulate will calculate all physics and update positions/velocities of balls

This involves calculating friction, checking to see if a ball has collided with the table and checking to see if a ball has collided with another ball The simulation rate can be changed by altering the simtime variable
