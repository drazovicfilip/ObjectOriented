Detailed Description

The Director will put together all needed objects into one big pool game. Will also parse JSON input.

Author
Filip Drazovic
Date
April 2018

In this case, there are multiple balls and a single table to be put in to the pool game. These will be built (using the builder) and put together using construct().

The director will also parse input from a given JSON file, named "config.JSON", specifying the game's table and ball(s) with all parameters.
