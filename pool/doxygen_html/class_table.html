Detailed Description

The Table is a concrete implementation of the abstract table, with nothing changed.

Author
Filip Drazovic
Date
April 2018

It is a static, drawn object that doesn't move during runtime. It holds the boundaries of the pool game which are used for colision calculations. These parameters should not change during runtime.
