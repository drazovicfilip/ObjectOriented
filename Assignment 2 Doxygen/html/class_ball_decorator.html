Detailed Description

The BallDecorator class uses the Decorator design pattern to add additional functionality to a given ball. Specifically, it will override the ball's draw method in order to draw more things - in this case, a cue which hits the ball. It holds a ball as a member variable, and all of its methods just call that ball's methods. For information about the method, have a look at the descriptions in ball.h.
